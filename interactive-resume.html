Interactive Resume

When applying for jobs, a common piece of advice is to fine-tune your resume for each job you are applying to. Recruiters often use ATS software to screen resumes before reading them, so applicants who make sure to use keywords found in the job description are more likely to get an interview. This interactive resume automatically fine-tunes itself based on keywords the user inputs. Try typing NumPy or CSS to see how my resume would differ for data scientist or full-stack developer jobs.
